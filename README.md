**Назначение**
Программа предназначена для построения таблицы истинности для заданного дерева разбора логического выражения.

**Входные данные**
Дерево разбора логического выражения (строка в формате ОПН).
Путь файла пользовательских функций.
Путь для сохранения результата работы программы.

**Выходные данные**
Таблица истинности выражения в формате cvs файла

**Описание**
Метод решения - Рекурсивное вычисление значения root-ноды методом Node::evaluate.
На C/C++ реализованы только базис логических операций:
    AND,OR,NOT
остальные операции задаются в файле пользовательских функций на основе базисных операций и других пользовательских функций.
Пользовательские функции вычисляются как поддеревья основного выражения с аргументами в виде значений её дочерних нод в основном выражении.
Для централизованной обработки вспомогательных задач используются:
    FunctManager - пользовательские функции (импорт, хранение).
    ErrorManager - обработка ошибок.
    ConfigManager - константы и ограничения (импорт, хранение).
